## 배열
리스트(list)는 알아도 배열(array)란 말이 생소할 수 있다.

## C언어의 배열(array)
프로그래밍에서 여러 개의 데이터를 관리하기 위한 구조들이 있다.
대표적으로 여기서 배울 배열(Array)을 배우겠지만
심화 공부 그리고 다른 프로그래밍 공부를 염두하여 ArrayList, LinkedList도 가볍게 살펴보기로 합시다.

특징들을 가볍게 살펴보면
**Array**
- 처음에 배열의 길이(=관리할 데이터의 최대 길이)를 정해야 합니다.
- 여러 개의 데이터를 관리하는 구조 중 특정 데이터 탐색하여 조회할 때(READ) 가장 빠릅니다. (**index**를 이용)
- 단, 데이터의 길이를 수정할 수 없는 단점이 크다. 새로 만들어야한다.

**ArrayList**
- Array와 구조는 유사하나 길이를 변경할 수 있다.
- 구조가 유사하다? -> 데이터 공간이 연속적이다.
- 데이터 길이를 수정할 수 없다는 점이 보완되었다.
- 데이터 추가 삭제시 메모리를 재할당하기 때문에 속도가 배열보다 느리다.

**LinkedList**
- 순서대로 늘어선 것이 아니라 자료의 주소 값으로 서로 연결되어 있는 구조
- 데이터 추가,삭제하기 좋다.
- 연속적인 데이터 공간의 확보가 필요없다. **포인터**로 이전 자료, 다음 자료를 연결하기 때문
- **포인터**로 인해 공간 낭비가 있다.
- 임의의 데이터에 접근할 때 시간이 오래걸린다. 

## 배열 기본 선언 방법들
#### 방법 1 : 선언과 할당을 따로
```c
// 선언 방법 : 타입 배열이름[길이];

// 배열 선언
int score[4];  // int형 배열 * 길이 4

// 값 할당(초기화), 길이 4 == 정상적인 index 0~3
score[0] = 100;
score[1] = 90;
score[3] = 90;

// 출력
printf("score 0번 값은 %d 입니다.\n", score[0]);
printf("score 1번 값은 %d 입니다.\n", score[1]);
```

#### (권장) 방법 2 : 선언과 초기화를 동시에
```c
// 배열 선언과 초기화
int totalScore = 0;
int score[4] = {100, 90, 80, 70};  // int형 배열 * 길이 4
int score[] = {100, 90, 80, 70};  // 우측에 초기화를 하면 배열 길이가 자동으로 계산된다.
```

## Array의 한계 다시보기
- 처음 생성부터 데이터 관리할 최대 길이를 정해야한다.
    - 너무 길다면 데이터 낭비
    - 너무 짧다면 삽입이 불가하니 다시 생성해야 됨
- 그런데 너무 스트레스 받지 말자.
    - 이제 막 입문한 여러분이라면 터무니 없는 수치만 아니라면 데이터 길이 때문에 스트레스 받을 일은 매우 적음
    - 충분히 넉넉한다고 생각한 길이를 넣고 사용하자.
