## 라이브러리
- 라이브러리의 개념과 종류: 표준 라이브러리, 외부 라이브러리 등
- 표준 라이브러리: stdio.h, string.h, math.h, time.h 등
- 라이브러리 함수의 기본 구조: 함수 이름, 매개변수, 반환값 등
- 주요 라이브러리 함수: printf, scanf, strcpy, strcat, pow, time 등
- string.h는 13강을 확인합시다

## 전처리기
- 전처리기는 C 프로그램을 컴파일하기 전에 코드를 변경하거나 추가하는 역할을 합니다.
- C 프로그램에서 #으로 시작하는 지시문들은 전처리기가 처리합니다.
- 전처리기는 이러한 지시문들을 기반으로 소스 코드를 변경하거나 다른 파일의 내용을 삽입하는 등의 작업을 수행합니다.
- 예를 들어
    - #define 지시문을 사용하면 상수를 정의할 수 있고, 
    - #include 지시문을 사용하면 다른 헤더 파일의 내용을 포함시킬 수 있습니다.
    - #ifdef 지시문과 같은 조건부 컴파일 지시문을 사용하면 특정 조건에 따라 코드를 컴파일하거나 무시할 수 있습니다.
- 전처리기는 코드 작성을 보다 쉽고 편리하게 만들어주는 역할을 하며, 컴파일러에게 전처리된 코드를 제공함으로써 오류를 줄일 수 있습니다.

### 전처리기 관련 키워드
- 전처리기 지시어: #include, #define, #ifdef, #ifndef, #endif 등
- #include 지시어: 다른 파일의 내용을 현재 파일에 포함시키는 역할
- #define 지시어: 매크로를 정의하여 코드에서 사용하는 상수나 함수처럼 활용하는 역할
- #ifdef, #ifndef, #endif 지시어: 조건부 컴파일을 위해 사용하는 역할

### 학습목표
C 언어에서 제공하는 math 라이브러리 함수와 time 라이브러리 함수를 이해하고 사용하는 방법을 학습한다.
전처리기의 개념과 기능을 이해하고, 전처리기를 사용하여 프로그램을 작성하는 방법을 학습한다.

### 개념잡기
math 라이브러리 함수: C 언어에서 제공하는 수학 함수를 모아놓은 라이브러리로, 수학적인 계산을 수행할 때 사용된다. 주요 함수로는 sin(), cos(), sqrt(), pow() 등이 있다.
time 라이브러리 함수: C 언어에서 제공하는 시간과 관련된 함수를 모아놓은 라이브러리로, 현재 시간을 구하거나 시간과 관련된 연산을 수행할 때 사용된다. 주요 함수로는 time(), localtime(), strftime() 등이 있다.
전처리기: C 언어에서 프로그램을 컴파일하기 전에 실행되는 단계로, 소스 코드를 처리하는 데 사용된다. 전처리기 지시문은 #으로 시작하며, #include, #define, #ifdef 등이 있다.

### 강의요약
math 라이브러리 함수
- math.h 헤더 파일을 포함하여 사용할 수 있다. 
- sin(), cos(), sqrt(), pow() 등의 수학 함수를 제공한다.

time 라이브러리 함수
- time.h 헤더 파일을 포함하여 사용할 수 있다. time() 함수를 이용해 현재 시간을 구할 수 있다
- localtime() 함수를 이용해 시간 정보를 분리하여 사용할 수 있다. 
- strftime() 함수를 이용해 시간 정보를 원하는 형식으로 출력할 수 있다.

전처리기
- #include 지시문을 이용해 헤더 파일을 포함하고, #define 지시문을 이용해 상수나 매크로를 정의할 수 있다. 
- #ifdef, #ifndef, #endif를 이용해 조건부 컴파일을 수행할 수 있다.
