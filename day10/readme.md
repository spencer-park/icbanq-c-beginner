## 함수 기본구조
```c
반환형 함수이름(){

    return 반환값;
}
```

## 반환 값이 없는 함수 -> void
```c
void 함수이름(){

}
```

## 반환 값이 있는 함수 -> 반환 값의 타입이 반환형
```c
int 함수이름(){

    return 반환 값;
}
```

```c
float 함수이름(){
    
    return 반환 값;
}
```

## 매개변수가 있는 함수
```c
int 함수이름(타입 매개변수명, 타입 매개변수명){

    return 반환 값;
}
```

## variable scope
1. 함수가 실행되고 종료되면, 그 안에 사용된 메모리는 정리된다.
2. 매개변수에는 값이 전달된다.
3. 매개변수에 주소값을 전달하면, 원본을 변경시킬 수 있다.(포인터 시간에)
